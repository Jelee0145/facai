# LLM 智能配置器 — 测试清单

> ⏳ 等待管理员在后台配置阿里云百炼 API Key 后执行

---

## 前置条件

- [ ] 管理员登录 `/admin` → `LLM 设置` 页面
- [ ] 填入阿里云百炼 API Key，选择模型 `qwen-vl-flash`
- [ ] 点击「测试连接」确认返回成功
- [ ] 点击「保存配置」

---

## 3. 自定义商品测试（LLM 已配置）— 核心验证

| # | 测试场景 | 操作 | 预期结果 | 状态 |

| 3.4 | **自定义"宠物窝"** | 添加"宠物窝" → 生成 | shot_type=product，LLM 推荐温馨家庭场景 | ⬜ |
